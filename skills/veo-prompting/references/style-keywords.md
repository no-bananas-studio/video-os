## Combining Keywords

Layer multiple keywords for precise control:

```
Cinematic, moody, cool blue tones with warm practical lighting,
film noir aesthetic, high contrast, slightly grainy 35mm look.
```

```
Dreamy, soft morning light, romantic soft focus, pastel color
palette, ethereal atmosphere.
```

```
Gritty documentary style, handheld feel, desaturated colors,
overcast diffused light, raw and authentic.
```
